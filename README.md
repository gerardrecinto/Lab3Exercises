# Lab 3 Exercises — Java

Java programming lab exercises from a university computer science course. Covers core OOP concepts, data structures, and algorithm implementations.

![Java](https://img.shields.io/badge/Java-8%2B-007396?logo=openjdk&logoColor=white)

## Contents

Exercises covering:
- Object-oriented design (inheritance, polymorphism, encapsulation)
- Collections: ArrayList, LinkedList, HashMap
- Algorithm basics: sorting, searching, recursion
- Exception handling

## Running

Compile and run any exercise with:

```bash
javac Exercise.java
java Exercise
```

Or import the project into IntelliJ IDEA or Eclipse.
